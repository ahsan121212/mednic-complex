# Mednic Complex Frontend

React Next.js 14 App Router application styled with Vanilla CSS and responsive design tokens.

## Scripts
- `npm run dev`: Starts the Next.js local dev server at `http://localhost:3000`.
- `npm run build`: Compiles production build.
- `npm start`: Starts production compiled build.

## Environment Config
- `NEXT_PUBLIC_API_URL`: Points to Express API base route (defaults to `http://localhost:5000/api` if blank).
