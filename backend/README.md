# Mednic Complex Backend

Node.js Express backend server storing clinical appointments and contact inquiries in MongoDB.

## Scripts
- `npm start`: Starts the production server.
- `npm run dev`: Starts the developer server with nodemon reloading.

## Environment Config
Rename `.env.example` to `.env` and fill out:
- `PORT`: Server Port (defaults to 5000)
- `MONGODB_URI`: MongoDB connection URI
